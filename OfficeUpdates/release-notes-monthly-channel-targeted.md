# <a name="release-notes-for-office-monthly-channel-targeted"></a>Office 릴리스 정보 월 단위 채널(대상 지정)

이 문서에는 Windows 데스크톱용 Word, Excel, PowerPoint, Outlook, Access 및 Project 의 참가자 빌드에 대한 릴리스 정보가 포함되어 있습니다. 매주 Microsoft는 흥미로운 새 기능, 중요한 수정 사항, 알리고자 하는 중요한 문제를 강조합니다. Microsoft는 일정 기간 동안 자주 참가자들에게 기능을 배포하며 때로는 수정 사항도 배포합니다. 이로서 더욱 광범위한 사용자들에게 기능을 릴리스하기 전에 원활하게 작동하는지 확인할 수 있습니다. 따라서 아래에 설명된 기능이 보이지 않더라도 곧 제공되게 되므로 걱정할 필요가 없습니다.  

> [!NOTE]
> - 릴리스 정보는 매주 게시되며 여러 빌드의 컴파일일 수 있습니다.
> - 릴리스 정보 게시 날짜는 실제 빌드 릴리스 날짜와 일치하지 않을 수 있습니다.
> - Office 365 ProPlus의 기존 설치에 대한 Microsoft Teams - 6월 말부터 Microsoft Teams가 Office 365 ProPlus(및 Office 365 Business)의 기존 설치에 업데이트 설치 시 포함됩니다. Teams가 추가되는 날짜는 사용 중인 업데이트 채널에 따라 다릅니다. 추가 정보는 [Office 365 ProPlus와 함께 Microsoft Teams 배포](https://docs.microsoft.com/ko-KR/deployoffice/teams-install)를 참조하세요.

### <a name="feature-updates"></a>기능 업데이트

### <a name="excel"></a>Excel

- ** 더 많은 접근 가능한 PDF 만들기:** PDF를 만들면 액세스 가능성 검사기가 저장하기 전에 수정해야 할 접근성 문제를 해결합니다. [자세한 정보](https://support.office.com/ko-KR/article/064625e0-56ea-4e16-ad71-3aa33bb4b7ed)

- **파일을 변환하여 접근성 향상:** 모든 사용자가 보다 쉽게 액세스할 수 있도록 파일을 최신 형식으로 업그레이드합니다.

- **민감도 레이블을 문서 및 전자 메일에 적용:** 조직의 정보 보호 정책을 계속 준수하기 위해 파일 및 전자 메일에 민감도 레이블을 적용합니다. [자세한 정보](https://aka.ms/officemipdocs)

### <a name="outlook"></a>Outlook

- **오류가 발생한 후 Outlook 업그레이드를 위한 메시지 표시:** Outlook에서 차후 버전에서 해결 된 오류가 발생하는 경우 Outlook은 고객이 업그레이드를 하고 수정 사항을 받도록 메시지를 표시합니다.

- **민감도 레이블을 문서 및 전자 메일에 적용:** 조직의 정보 보호 정책을 계속 준수하기 위해 파일 및 전자 메일에 민감도 레이블을 적용합니다. [자세한 정보](https://aka.ms/officemipdocs)

### <a name="powerpoint"></a>PowerPoint

- ** 더 많은 접근 가능한 PDF 만들기:** PDF를 만들면 액세스 가능성 검사기가 저장하기 전에 수정해야 할 접근성 문제를 해결합니다. [자세한 정보](https://support.office.com/ko-KR/article/064625e0-56ea-4e16-ad71-3aa33bb4b7ed)

- **파일을 변환하여 접근성 향상:** 모든 사용자가 보다 쉽게 액세스할 수 있도록 파일을 최신 형식으로 업그레이드합니다.

- **민감도 레이블을 문서 및 전자 메일에 적용:** 조직의 정보 보호 정책을 계속 준수하기 위해 파일 및 전자 메일에 민감도 레이블을 적용합니다. [자세한 정보](https://aka.ms/officemipdocs)

### <a name="word"></a>Word

- **공동 작성 기능 개선**: 문서를 공동 작성 시 변경 내용 추적 기능을 사용하여 Word의 성능을 개선했습니다.

- ** 더 많은 접근 가능한 PDF 만들기:** PDF를 만들면 액세스 가능성 검사기가 저장하기 전에 수정해야 할 접근성 문제를 해결합니다. [자세한 정보](https://support.office.com/ko-KR/article/064625e0-56ea-4e16-ad71-3aa33bb4b7ed)

- **파일을 변환하여 접근성 향상:** 모든 사용자가 보다 쉽게 액세스할 수 있도록 파일을 최신 형식으로 업그레이드합니다.

- **3D 개체의 병합으로 발생하는 문서 손상 문제를 해결했습니다.:** 3D 개체의 병합으로 발생하는 문서 손상 문제를 해결했습니다.

- **민감도 레이블을 문서 및 전자 메일에 적용:** 조직의 정보 보호 정책을 계속 준수하기 위해 파일 및 전자 메일에 민감도 레이블을 적용합니다. [자세한 정보](https://aka.ms/officemipdocs)

- **Office 365 ProPlus의 기존의 설치에 Microsoft Teams를 설치** 6월말부터 Microsoft Teams가 Office 365 ProPlus(및 Office 365 Business)의 기존 설치에 업데이트를 설치 시 포함됩니다. Teams가 추가되는 날짜는 사용 중인 업데이트 채널에 따라 다릅니다. 추가 정보는 Office 365 ProPlus에 Microsoft Teams 배포를 참조하세요. [자세한 정보](https://docs.microsoft.com/ko-KR/deployoffice/teams-install)

### <a name="resolved-issues"></a>해결된 문제
### <a name="access"></a>Access

- 사용자가 공유 데이터베이스를 사용하는 경우 "일관성이 없는 상태" 오류가 표시되는 문제를 해결했습니다.

### <a name="excel"></a>Excel

- 함수를 동기적으로 실행하게 만드는 비동기적인 사용자 정의 함수의 성능 문제를 해결했습니다.
- 이전 버전의 Office에서 작성한 통합 문서를 현재 버전의 Office에서 열었을 때 Excel이 중단될 수 있는 문제를 해결했습니다.
- 사용자가 일부 보호 시트에 하이퍼링크를 붙여넣지 못하게 할 수 있는 문제를 해결했습니다.
- 리본에 있는 글꼴 이름이 사용중인 글꼴과 다를 수 있는 문제가 해결되었습니다.
- 일부 콘텐츠에 하이퍼링크 서식이 제대로 적용되지 않는 문제를 해결했습니다.
- 구조적 절대 참조를 포함하는 수식이 잘못 조정될 수 있는 문제를 해결했습니다.
- Excel에서 복사한 콘텐츠를 다른 Office 응용 프로그램에 붙여 넣을 시 정확히 표시되지 않는 문제를 해결했습니다.
- 시리즈 컬렉션을 변경할 때 분산형 차트가 제대로 렌더링되지 않을 수 있는 문제를 해결했습니다.
- Excel을 시작할 때 가끔 중단되는 문제를 수정했습니다.
- OneDrive에서 파일을 개체로 삽입하는 경우 발생하는 문제를 해결했습니다.
- 공동 작성 세션이 진행되는 동안 피벗 테이블을 새로 고칠 수 없었던 문제를 해결했습니다.
- 인쇄 미리 보기의 인쇄 영역이 올바르지 않았던 문제를 해결했습니다.
- 차트 템플릿을 사용하여 차트를 삽입할 때 미리 보기에 사용되는 색을 수정했습니다.

### <a name="onenote"></a>OneNote

- 복사한 페이지로의 링크가 새로 복사한 페이지가 아닌 원본 페이지로 이동시킬 수 있는 문제를 해결했습니다.

### <a name="outlook"></a>Outlook

- Internet Explorer에서 제한된 사이트에 대해 보호 모드가 해제되었을 때 Outlook에서 부적절한 리소스 사용을 일으킬 수 있는 문제를 해결했습니다.
- Office 365에서 최신 인증으로 마이그레이션할 때 Outlook에 여러 메일함이 있는 고객의 메일함 동기화를 방지할 수 있는 문제를 수정했습니다.
- 드롭다운 메뉴에 서명 레이블의 일부 문자가 표시되지 않는 문제를 수정했습니다.
- 사용할 수 있는 회의실 수가 많은 경우 회의실 선택 성능을 크게 개선하였습니다.
- 사용자가 일정에 첨부 파일을 추가하지 못하도록 하는 문제를 해결했습니다.
- 디지털 서명된 메시지에 회신할 때 오류 메시지가 표시되는 문제를 해결했습니다.
- 검색 결과에 확장된 비행 결과 검색이 표시되지 않도록 하는 문제를 해결했습니다.
- 간단한 보기에서 UI가 정지되는 문제를 해결했습니다.
- 일부 사용자가 그룹 일정 보기에서 오프라인으로 잘못 표시되는 문제를 해결했습니다.
- ANSI 원본에서 텍스트를 붙여넣을 때 유니코드 문자가 나타날 수 있는 문제를 해결했습니다.
- 메일 폴더의 복제를 발생시킬 수 있었던 문제를 수정했습니다.
- S/MIME 암호화 전자 메일을 보내려고 할 때 잘못된 오류 메시지를 발생시킬 수 있었던 문제를 수정했습니다.
- 사용자가 Skype에서 '부재 중 대화' 메시지를 받을 시 때때로 충돌이 발생할 수 있는 문제를 수정했습니다.
- 메모리 누수를 발생시킬 수 있는 문제를 수정했습니다.
- 공유 일정 폴더와 상호 작용하는 경우에 사용 권한 오류가 보고될 수 있는 문제를 해결했습니다.
- 초안으로 저장할 때 보낸 사람 이름을 변경할 수 있었던 문제를 수정했습니다.

### <a name="powerpoint"></a>PowerPoint

- 사용자가 PDF로 인쇄할 때 오류가 발생할 수 있는 문제를 해결했습니다.
- AirSpace WinComp에서 실행 시 ARC 장치의 연결이 끊어질 수 있는 문제를 확인했습니다.
- 하이퍼링크를 사용하여 텍스트를 붙여넣을 때 하이퍼링크를 만들지 못하게 하였던 문제를 수정했습니다.
- 통계가 제대로 작동하지 않도록 하였던 문제를 수정했습니다.
- 슬라이드 마스터 및 슬라이드 레이아웃에서 머리글/바닥글/슬라이드 번호 플레이스홀더에 텍스트를 붙여 넣은 후 TextRanges를 중단시킬 수 있었던 문제를 수정했습니다.

### <a name="project"></a>Project

- 엔터프라이즈 리소스를 로컬 리소스로 바꿀 때 충돌이 발생하는 문제를 수정했습니다.
- 프로젝트가 마스터 프로젝트를 통해 Project Web App에 저장될 때 하위 프로젝트 로컬 리소스, 요율 및 기타 속성이 저장되지 않던 문제를 해결했습니다.
- 보호된 작업을 사용하도록 설정한 경우 요약 작업의 작업 값을 변경하면 충돌이 발생할 수 있는 문제를 해결했습니다.
- 마스터 프로젝트가 있고 마스터 프로젝트의 로컬 사용자 정의 필드 값을 복사하여 삽입된 프로젝트에 붙여넣는 경우 일부 정보가 제대로 저장되지 않던 문제를 해결했습니다.
- 이벤트를 엔터프라이즈 일정과 동기화할 수 없는 문제를 해결했습니다.

### <a name="word"></a>Word

- Ctrl + v 단축키가 손상될 수 있는 문제를 해결했습니다.
- 초안 보기에서 동기식 스크롤링이 제대로 작동하지 않는 문제를 수정했습니다.
- Deskjet 프린터로 인쇄할 때 배율 문제가 발생할 수 있는 문제를 해결했습니다.
- 처음으로 문서를 저장한 후 도구 조언이 제대로 표시되지 않는 문제를 해결했습니다.
- 표 서식이 손실될 수 있는 문제를 해결했습니다.
- OneDrive에서 파일을 개체로 삽입하는 경우 발생하는 문제를 해결했습니다.
- 글꼴 색이 적용되지 않았던 문제를 수정했습니다.

### <a name="office-suite"></a>Office 제품군

- 보류 중인 업로드가 있는 파일을 닫으면 사용자에게 잘못된 오류 메시지가 나타날 수 있는 문제를 해결했습니다.
- 파일의 로컬 버전을 삭제하라는 경고가 반복될 수 있는 문제를 해결했습니다.
- 해당 기능을 사용하지 않도록 설정한 경우 버전 기록을 제공할 수 있었던 문제를 해결했습니다.
- Outlook 기본 인증 프롬프트에 로그인 이름이 이제는 미리 채워집니다.
- 보통 굵기의 텍스트에 잘못된 스타일이 적용되는 문제를 해결했습니다.
