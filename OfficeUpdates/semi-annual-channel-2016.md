# <a name="release-notes-for-semi-annual-channel-releases-in-2016"></a>2016 년 반기 채널 릴리스에 대 한 릴리스 정보

이러한 릴리스 정보에서는 2016의 Office 365 ProPlus에 대 한 반기 채널 업데이트에 포함 된 새로운 기능, 보안 업데이트 및 비보안 업데이트에 대 한 정보를 제공 합니다. 
> [!NOTE]
> - 또한 Visio Pro for Office 365 및 Project Online 데스크톱 클라이언트에 대한 새로운 기능, 보안 업데이트 및 비보안 업데이트에 대한 정보도 제공합니다.
> - 이 정보는 일부 Office 365 계획에 제공되는 Office 버전(예: Business Premium)인 Office 365 Business에도 적용됩니다.
> - 반기 채널이 2018년 1월 이전에는 지연 채널로 지칭되었습니다.

### <a name="excel-non-security-updates"></a>Excel: 비보안 업데이트
-   백 엔드 서버에 대한 인증을 위해 암호 프롬프트를 요구하는 연결된 ODBC 테이블을 참조하는 Access 쿼리에 연결하려고 할 때 암호 프롬프트 대신 오류가 발생하는 64비트 버전의 Excel 문제를 해결합니다.
-   OLAP 드릴스루 동작으로 인해 Excel이 작동 중지되는 문제를 해결합니다.
-   많은 텍스트 상자가 포함된 통합 문서를 편집하려고 할 때 Excel이 응답하지 않는 문제를 해결합니다.
-   최근 보안 업데이트로 인한 문제를 해결합니다. 이 문제로 인해 신뢰할 수 없는 위치에서 HTML 파일을 열 때 제한된 보기에서 파일을 열 수 있도록 허용하는 대신, 오류가 표시되지 않는 빈 화면으로 Excel이 열립니다.
-   OLAP 데이터 원본을 새로 고치면 1분 후에 시간이 초과되는 문제를 해결합니다.
-   Application.OnUndo를 사용하여 프로그래밍 방식의 평준화 취소를 포함하는 통합 문서를 저장할 때 통합 문서의 마지막 변경 내용이 실행 취소되는 문제를 해결합니다.
-   Excel이 일부 상황에서 셀 값 변경을 내레이터에 알리지 않는 문제를 해결합니다.
-   여러 워크시트를 포함하는 통합 문서에서 데이터 유효성 검사 드롭다운 목록이 손상되고 자체적으로 복제되는 문제를 해결합니다.
-   컴퓨터에서 Office 이전 버전을 제거할 때 Excel에 대한 파일 연결이 손실되는 문제를 해결합니다.
-   파일을 저장한 후에 Excel을 닫아도 임시 파일이 그대로 남아 있는 문제를 해결합니다.
-   삭제된 워크시트에 대해 메서드나 속성을 호출할 경우 해당 워크시트에서 개체를 찾을 수 없다는 오류라 표시되지 않고 Excel이 중단되는 문제를 해결합니다.
-   Excel을 열었을 때 시작 화면이 나타나지 않도록 구성했는데도 추가 기능이 사용하도록 설정되면 시작 화면이 나타나는 문제를 해결합니다.
-   데이터 \> 가져오기 및 변환 \> 테이블에서를 사용할 때 명명된 범위를 올바르게 가져오지 않는 문제를 해결합니다.
-   원격 분석 대시보드에 대해 로깅이 사용하도록 설정된 경우 컴퓨터에서 매크로가 현저히 느리게 실행되는 문제를 해결합니다.
-   부분합을 만든 후 선택한 범위가 부분합을 포함하도록 확장되지 않는 문제를 해결합니다.
-   FTP 서버의 파일을 열 수 없는 문제를 해결합니다.

### <a name="outlook-non-security-updates"></a>Outlook: 비보안 업데이트
-   사용자 지정 편지지를 기본 테마로 사용할 경우 새 전자 메일 메시지에 필요 없는 줄 바꿈이 추가되고 글꼴이 Calibri로 설정되는 문제를 해결합니다.
-   Windows 디스플레이가 125%로 설정되어 있을 때 Outlook을 열면 Outlook이 중단되는 문제를 해결합니다.
-   내레이터 등의 Microsoft UI 자동화 수신기를 실행 중에 모임 요청을 보면 Outlook의 작동이 중단되는 문제를 해결합니다.
-   MAPI/HTTP에 사용할 수 있는 Exchange Server 2013에 연결할 경우 사용자의 데스크톱 자격 증명을 사용하여 자동으로 로그인되는 대신 자격 증명을 묻는 메시지가 표시되는 문제를 해결합니다.
-   폴더의 PST 파일에 전자 메일 메시지를 끌어서 놓으면 Outlook에서 파일 복구를 실행하도록 요구하는 문제를 해결합니다.
-   Outlook 2016에서 보낸 모임 초대가 Outlook 2010 또는 Outlook 2013에서 수신될 경우 초대의 이미지가 누락되는 문제를 해결합니다.
-   높은 DPI를 사용할 때 표를 포함하는 메시지의 모양이 향상됩니다.
-   파일 \> 옵션 \> 메일 \> 편지지 및 글꼴에서 "회신 또는 전달 시 새로운 색 선택"을 선택한 경우에도 회신 또는 전달 시 새 색이 선택되지 않고 검은 글꼴 색이 사용되는 문제를 해결합니다.
-   곧 발생할 것으로 보이는 PST 손상의 감지 능력을 높이고 손상 방지를 위한 단계를 추가합니다.
-   .rtf 형식으로 전자 메일을 보낼 때 확장 기호 문자가 “1”로 변경되는 문제를 해결합니다.
-   받은 편지함 또는 지운 편지함 폴더와 같은 폴더의 메시지 목록 맨 아래에 서버의 이전 메일 메시지를 볼 수 있는 링크가 나타나지 않는 문제를 해결합니다.
-   .msg 파일을 사용하여 메일을 보낼 때 Outlook에서 메시지 콘텐츠를 보내지 않는 문제를 해결합니다.
-   받은 편지함 오류 수정 도구(scanpst.exe)가 .pst 파일의 인덱싱 및 검색 방법과 관련된 파일 손상 문제를 감지하고 수정할 수 있도록 이 도구를 업데이트하세요. 이러한 유형의 손상이 발생하면 .pst 파일이 로드되지 않거나, Outlook이 충돌하거나, CPU 사용률이 높아질 수 있습니다.
-   일반 텍스트 메시지에 메시지 분류가 일부 표시되지 않는 문제를 해결합니다.
-   대화 설정 \> 들여쓴 클래식 보기 사용 옵션이 사용하도록 설정된 경우 Outlook 검색이 중복된 결과를 반환하는 문제를 해결합니다.
-   스페인어 버전의 Outlook에서 새 메시지를 만들 때 필드의 탭 탐색 문제를 해결합니다.
-   새 메시지를 보내거나 회신을 할 때 HTML 서명의 이미지가 회색 상자로 나타나는 문제를 해결합니다.
-   읽은 항목에 검색 폴더에 읽지 않은 항목이 표시되는 문제를 해결합니다.
